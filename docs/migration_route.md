# 🌍 Route Migratoire Climatique — Colonie AMI
## Cycle complet : 63 jours = 1 année colonie

> La zone climatique se déplace en suivant la rotation terrestre.
> **Hémisphère Nord** → Ouest vers Est
> **Hémisphère Sud** → Est vers Ouest
> Transitions douces entre hémisphères — pas de saut > 100km impossible.
> Météo réelle via **Open-Meteo API** (gratuit, sans clé).

## 🔧 Implémentation technique

### Open-Meteo API — appel type
```javascript
const url = `https://api.open-meteo.com/v1/forecast
  ?latitude=${city.lat}
  &longitude=${city.lon}
  &current=temperature_2m,
    wind_speed_10m,
    precipitation,
    weather_code,
    visibility`

const response = await fetch(url)
const data = await response.json()
```

### Schéma table `cities` Supabase
```sql
create table cities (
  id            serial primary key,
  nom           text not null,
  pays          text not null,
  latitude      decimal(6,3) not null,
  longitude     decimal(6,3) not null,
  hemisphere    char(1) check (hemisphere in ('N','S')),
  ordre         integer unique not null,
  phase         text check (phase in ('nord','transition_ns','sud','transition_sn')),
  actuelle      boolean default false,
  distance_km   integer  -- distance depuis ville précédente
);
```

### Schéma table `weather_log` Supabase
```sql
create table weather_log (
  id            serial primary key,
  city_id       integer references cities(id),
  temperature   decimal(4,1),
  vent_kmh      decimal(5,1),
  precipitation decimal(5,1),
  raw_data      jsonb,
  event_id      integer references events(id),
  created_at    timestamptz default now()
);
```

### pg_cron — cycle météo toutes les 12h
```sql
select cron.schedule(
  'weather-cycle',
  '0 */12 * * *',
  $$ select run_weather_cycle() $$
);
```
